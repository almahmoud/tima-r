You can use this script with the following example:
Rscript inst/scripts/prepare_features_classification.R --input 'inst/extdata/interim/annotations/a177b17ded9f4e01bf343317bfc0372e_isdb_filled.tsv.gz' --output 'inst/extdata/interim/annotations/a177b17ded9f4e01bf343317bfc0372e_isdb_treated.tsv.gz' --library 'inst/extdata/interim/libraries/library.tsv.gz' --quickmode TRUE

Usage: prepare_features_classification.R [--input=<input>] [--output=<output>] [--library=<library>]

Arguments:
  -i --input=<input>            Your isdb result file. Supports compressed files.
  -o --output=<output>          Path and filename for the output. If you specify .gz file will be compressed.
  -q --quickmode=<quickmode>    Boolean. Do you want to classify only with LOTUS (quick) or submit to GNPS (long)?
  -t --tool=<tool>              Tool used for generating the component ids, currently only GNPS

Options:
  -h --help                     Show this screen.
  -v --version                  Show version.
