You can use this script with the following example:
Rscript R/fill_chemical.R --input '../data/interim/b988e66d1542430cbc6e703be781e49c_isdb_pretreated.tsv.gz' --output '../data/processed/b988e66d1542430cbc6e703be781e49c_isdb_treated.tsv.gz' --quickmode TRUE

Usage: fill_chemical.R [--input=<input>] [--output=<output>] [--quickmode=<quickmode>]

Arguments:
  -i --input=<input>          Your isdb result file. Supports compressed files.
  -o --output=<output>        Path and filename for the output. If you specify .gz file will be compressed.
  -q --quickmode=<quickmode>  Boolean. Quickmode only searches against LOTUS locally, otherwise might take long (depending on your number of candidates with missing information)

Options:
  -h --help         Show this screen.
  -v --version      Show version.
